# 🏨 Hotel Landing Page

An elegant and sophisticated landing page for a luxury hotel and resort.

## 🚀 Features

- Room showcase
- Booking system layout
- Amenities presentation
- Virtual tour gallery
- Special offers section
- Restaurant menu
- Event spaces

## 💻 Technologies

- HTML5
- CSS3/SCSS
- JavaScript ES7+
- Bootstrap
- Responsive Design

## 🛠️ Installation

1. Clone the repository
```bash
git clone https://github.com/Khadalicioso/Websites.git
```

2. Navigate to project directory
```bash
cd "Hotel Landing Page"
```

3. Open in your preferred browser
```bash
index.html
```

## 📱 Responsive Design

- Mobile-first approach
- Tablet & desktop optimized
- Cross-browser compatible
- High-resolution images
